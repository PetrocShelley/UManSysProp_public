Contact

Academic enquiries on provided functionality, existing and future work

UManSysProp was designed to better facilitate collaborations across multiple sectors and disciplines. If you have any questions or ideas regarding future development work, or you simply want to work together, please contact Dr David Topping. The source code for the predictions covered on this site is not to be made available. As noted on the homepage however, you can access these predictions outside of your web-browser using a programmer friendly JSON API that enables you to call our suite of tools from your own code.
